Chap.3 エネルギー、触媒作用、生合成
	
	細胞内で起こる化学反応
		異化 ... 取り込んだ物質を細かい小分子に分解する
		同化 ... 小分子から細胞構築用の大きい分子を構築する
		合わせて細胞の代謝（metabolism）という。
	
	秩序形成のためにはエネルギーが必要
		熱力学第二法則 ... 閉鎖系のエントロピー増大
		生物は解放系で、外部から自由エネルギーを取り込んでエントロピーを一定に抑えている
		
	エネルギー源
		熱力学第一法則 ... 結合エネルギー -> 自由エネルギー
		食物連鎖 ... 最底辺の植物は日光からエネルギーを得ている
		＝光合成（電磁エネルギー→化学エネルギー）
	
	光合成
		明反応 ... 日光によるATP生成
		暗反応 ... ATPによる糖生産（炭素固定）
	呼吸
		有機分子を酸化させて共役した反応でATPを生産する
	
	酸化還元
		酸化（oxidation） ... 電子を奪う
		還元（reduction） ... 電子を与える
	
	酵素（enzyme）
		触媒（catalyst）作用
			反応の活性化エネルギーを小さくする。反応熱を変化させる事は無い
		酵素と反応する分子を基質（substrate）という
		酵素の活性部位が特定の分子との反応を担保している
		複数の酵素が反応し得る基質の場合、酵素の存在によって反応経路を確定出来る
		拡散（diffusion）のお陰で基質と酵素は反応する事が出来る
	
	共役
		熱力学的に有利な反応と不利な反応を共役させ、秩序を形成する
		（自由エネルギー変化ΔGが正の反応を可能にする）
		例えば、エネルギー的に不利なX->Yと、有利なY->ZがあるとYが常に不足気味になるので反応が進む。
	
	自由エネルギー変化
		標準自由エネルギー変化（ΔG°）
		ΔG = ΔG° + 0.616ln[X]/[Y]　（Y->Xの反応）
		平衡定数 K=[X]/[Y]
		平衡状態：濃度効果とΔG°の反応推進力が釣り合う点
			X+Y -> XY反応だと、結合エネルギーが大きいほどにKは大きくなる
	
	酵素反応
		VmaxとKmが酵素の強さに関する代表的なパラメータ
		KmはV=1/2Vmaxなる基質濃度を示す。小さいほどに強い。
		酵素反応速度に影響する要因
			基質濃度、拮抗剤（antagonist）、生成物濃度など
		
	活性型運搬体分子（エネルギー通貨）
		ATP, NADH, NADPHが代表的
		高エネルギー結合（リン酸結合）を持ち、加水分解で熱を放出する
		酵素による共役で熱を有効に利用する事で不利な反応を起こす事が出来る
		
		ATP = ADP + Pi
			大抵の共役はリン酸化を含む。
			ATPによる二分子の結合の促進 A-H + B-OH -> A-B + H2O
				B-OH + ATP -> B-O-PO3 + ADP
				A-H + B-O-PO3 -> A-B + Pi
		
		NAD+（nicotinamide adenine dinucleotide）, NADP+（nicotinamide adenine dinucleotide phosphate）
			電子とH+を受け取って還元型（NADH, NADPH）となる
			電子（あるいはヒドリドイオン）の運搬体としてみなせる
			両者電子保持の仕組みはほぼ同じだが、Pi一つ分の分子構造の違いにより、関係する酵素に差がある
			NADHは主に異化反応系の中間体、NADPHは主にどうか反応を触媒する酵素とともに働く
		
		その他の活性型運搬体分子
			CoA : アセチルCoAとしてアセチル基を運ぶ
			カルボキシル化ビオチン：カルボキシ基を運ぶ->オキサロ酢酸の合成（クエン酸回路）
			ヌクレオチドを含む事が多く、RNAワールドの痕跡と考える事が出来る
			色々な基の運搬のための運搬体分子が存在するが、基本的にはもとをただせばATPがエネルギー源
		
		ATP->ADP+Piで得られるエネルギーはΔG=-13kcal/mol
			これで足りない反応は、サブ反応カスケードを作るか
			AMP+ピロリン酸分解により26kcal/molを得て起こすことになる。
