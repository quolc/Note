Chap 14. ミトコンドリアと葉緑体におけるエネルギー生産
	
	細胞はそのエネルギーの大半を膜に配置された系を使って得ている
		解糖系はごく一部。
		1. 電子伝達系でプロトン輸送
		2. H+の勾配による移動時にATP合成酵素が働く
		化学浸透共役
	
	ミトコンドリアと酸化的リン酸化
		ほとんどの真核細胞に広く存在する
		1グルコースから30ATPを生産できる。（解糖系の15倍）
		独自のDNA, RNA, 転写翻訳系を持っていて自身のタンパク質の一部を合成できる
		１細胞中に複数個存在する（多いと1000個単位）
			
		外膜、内膜と２つの内部区画を持つ
			内側の大きい空間がマトリックス、狭い部分は膜間部分と呼ばれる
			外膜
				ポリンという輸送タンパクを多数含む、親水性の大きなチャネルを形成する
			内膜
				膜輸送タンパクにより運ばれるもの以外ほとんど何も通さない
				電子伝達とプロトン汲み出しを行う場、ATP合成も行われる
				複雑に折りたたまれてひだ状の構造をとっている（クリステ）
	
		クエン酸回路
			ピルビン酸や脂肪酸をマトリックスでアセチルCaAに変換し、クエン酸回路に入る
			 -> NADH, FADH2が生産され、CO2が排出される
		
		化学浸透課程
			クエン酸回路で生産されたNADH, FADH2
			 -> ミトコンドリア膜にある電子伝達系に高エネルギー電子を与える（再生される）
			 -> 電子伝達系を経由してO2に達し、H2Oが生産される
			電子伝達系 ... 高エネルギー電子のエネルギーを利用してH+を膜間へ汲み出す
			形成されたプロトン勾配がATP合成を駆動する（酸化的リン酸化）
		
		電子はミトコンドリア内膜にある一連のタンパク質を経て運ばれる
			電子伝達系 - 呼吸鎖とも
			呼吸鎖は三種類の呼吸酵素複合体で主に構成される
			1. NADH脱水素酵素複合体
			2. シトクロムb-c1複合体
			3. シトクロム酸化酵素複合体

		ATP合成酵素
			H+汲み出しによってプロトン濃度は10倍程度の差が生じる
			ミトコンドリア内膜に埋め込まれており、H+通過時に柄が頭部の中で回転してATPを合成する
			モーターとしても発電機としても動く。ATP合成も加水分解もする
			1ATPにつき3プロトンが消費される
			
		ミトコンドリア内膜を通す共役輸送は電気化学的プロトン交配によって駆動される
			ATP合成以外も電気化学的プロトン勾配を利用する
			輸送タンパク
			最近の鞭毛の回転もプロトンモーターで駆動される
		
		細胞内のATP:ADP比は常に高く保たれている
			ATP 10 : 1 ADP くらい
		
	電子伝達系の詳細
		プロトン ... 水の形で細胞内のあらゆる場所に偏在している, 媒体として向いている
		酸化還元電位 = 電子に対する親和性
			NADHは酸化還元電位が低い = エネルギーが高い
			酸素の還元と共役すると⊿G=-52.4kcal/mol(NADH)で、ATP合成のための+7.3kcal/molをはるかに上回る
		
		ユビキノン ... 還元されて電子を二重層内で運搬する
		その他の電子運搬体は金属を含むことが多い（シトクロムとか）
		呼吸鎖の最後には鉄や銅の原子に至る（シトクロム酸化酵素複合体）
			O2を直接還元するとスーパーオキシドラジカルが生じて危険
			酸素分子を保持して還元を受けきるのに十分な電子が揃うまで待機する
	
		呼吸は効率が高い
			変換効率が40%くらいに達する ... 電気モーターやガソリンエンジンの倍程度の効率
			
	葉緑体と光合成
		葉緑体はミトコンドリアと似ている
			二枚の膜構造による
			ミトコンドリアのマトリックス -> ストロマ
		違い
			内膜には電子伝達系が含まれていない
			チラコイド膜の存在、ここに光捕捉系、電子伝達系、ATP合成酵素がある
			チラコイド膜が層状に積み重なった構造をグラナという。
			チラコイド膜は互いに繋がって閉じたトポロジーを持ってる
		
		光合成の反応
			1. 光合成電子伝達反応（明反応）
				クロロフィルを活性化してチラコイド膜の電子伝達系でH+をチラコイド膜に汲み入れる
				NADPHが生産される, 副産物としてO2も出る
			2. 炭素固定反応
				NADPHのエネルギーを使ってCO2を炭水化物に変換する
				ストロマで始まって細胞質でも反応が続く。
		
		明反応
			クロロフィルが光を吸収して励起状態になる
			光化学系を構成し、アンテナがクロロフィルによってしめられている
			反応中心へと励起電子のエネルギーが渡され、スペシャルペアとなるクロロフィル電子を励起する
			反応中心では高エネルギー電子をクロロフィルから引き離し安定した環境に送り込む
			光のエネルギーはまず光化学系IIに吸収されて、高エネルギー電子の生産に使われる
			光化学系Iへと電子が受け渡される過程でH+が汲み入れられ、勾配が作られる
			光化学系Iもまた独自に光を吸収し、プラスとシアニンを電荷分離する
		
		炭素固定
			CO2から有機炭素の作られる反応
			リブロースビスリン酸カルボキシラーゼ（ルビスコ） ... CO2を取り込んで五炭糖（リブロース1,5-ビスリン酸）と反応させる
			炭素固定回路（カルビン回路）を経由して3分子の二酸化炭素から1分子のグリセルアルデヒド3-リン酸が作られる
			この過程で3ATP + 2NADPHが消費される
		
			グリセルアルデヒド3-リン酸は解糖系でピルビン酸に変換されてからATP合成に用いられる
			スクロースなどの代謝産物にも変換される。ほとんどはデンプンに
	
	ミトコンドリアと葉緑体の起源
		両者とも、10億年移譲前に古代の真核細胞に飲み込まれた最近が進化したものか
		... 独立した転写系などの器官やゲノムをもつ
		ほとんどの遺伝子は核内に移転しているので、ごく少数の遺伝子しかミトコンドリア内には保持さrていない
		葉緑体も同様だが、より多くの遺伝子を持っている
		
		進化の予想
			当初 ... 環境中のグルコースなどの水素を含む分子からのエネルギーの取り出し
			 -> 老廃物として有機酸が排出され、環境のpHが低下した
			H+ポンプの有用性が生じる
			 -> H+ポンプの発達、それによるATP合成の主流化
			H+汲み出しにATPを消費しないシステムや、有機酸から電子を取り出せるシステムの適応度が高くなる
			 -> 現在の電子伝達系へ
			
			光合成は二段階に分けて光化学系が獲得された
			一回目はおそらくH2SからNADPHに水素原子を移動させるもの ... 環境のH2Sに依存
			二回目で大きなエネルギーが取り出せるようになり、H2OをH+源として使えるようになった -> 環境依存性が低下した
			
			酸素の存在によって好気的代謝によるATP合成を行う最近が出現できるように
			呼吸依存性の細菌を真核細胞が飲み込んだところでミトコンドリアが生じた
			
		メタン細菌
			原始的な環境に生育、古細菌の一種
			噴出口からの水素や窒素ガスだけを利用して無機栄養だけで生育する
			
			1. 窒素固定 ... 窒素分子を還元してアンモニアを得る
			2. エネルギー獲得 ... H2からCO2への電子伝達 -> 副産物としてのメタン
			この過程でプロトン勾配が形成され、ATPが合成される
			
			独自の炭素固定方法
			H2がCO2のCOへの還元に用いられる, ついでアセチルCoAに
