Chap.4 タンパク質の構造と機能
	
	☆左にN末端、右にC末端を書く
	
	タンパク質の機能例
		酵素 ... DNAポリメラーゼ、各種キナーゼなど
		構造タンパク質 ... チューブリン、コラーゲン、ケラチンなど
		輸送タンパク ... ヘモグロビン、Ca2+ポンプなど
		モータータンパク ... ミオシン、アクチン、キネシンなど
		貯蔵タンパク ... カゼイン（アミノ酸）、フェリチンなど
		シグナルタンパク ... インスリン、上皮増殖因子など
		受容体タンパク ... ロドプシン、インスリン受容体など
		遺伝子調節タンパク ... ラクトースリプレッサー、ホメオドメインタンパクなど
		特殊な機能を持つタンパク ... GFP, 凍結防止タンパクなどなど
	
	タンパク質＝アミノ酸ポリペプチド
		側鎖の並びがタンパク質の性質を決める
		側鎖同士の間に働く非共有結合 : 水素結合、イオン結合、ファンデルワールス力
		折りたたみを受けて立体構造が形成される
		・非極性側鎖は内側にしまわれる
		・極性分子は外側で溶媒の水と水素結合
		
	タンパク質はエネルギー最小のコンフォメーションに折り畳まれる
		溶媒に他の物質を大量に入れたりする -> 変性
		溶媒を通常状態に戻す -> 再生
		折りたたみは自己組織的に起こる。（シャペロンによって促進される）
	
	タンパク質の形
		アミノ酸30-10000程度まで様々なサイズがある。50-2000が一般的
		構造解析の第一歩はアミノ酸配列の決定
	
	タンパク質構造のパターン
		二次構造
			αヘリックス
				らせん構造は典型的（周期的なパターンがらせんを作る）
				二本のαヘリックスが巻き付いて「より合わせコイル」を作る事もある
			βシート
				平行βシート、逆平行βシートが存在する
				強固なひだ状の構造を作り、多くのタンパク質のコアとなる
		三次構造
		四次構造　という階層が基本構造
			大型タンパク分子は複数のポリペプチド鎖が結合部位で互いに密接にくっついて多量体を構成している
			ヘモグロビンが典型的
	
	タンパクドメイン : 一次〜四次構造とは異なる切り口の構造
		複数のタンパクで共通する、他の部分から独立して密に折り畳まれる領域
		100-250proteinで構成される
		機能モジュールに対応
	
	ポリペプチドには莫大なアミノ酸の組み合わせが存在する
		安定な三次元構造を取れるのはそのうちのごく一部
		進化の過程で特に安定な物が多く残された
		事実、酵母α2タンパクとショウジョウバエのengrailedタンパクのDNA結合ドメインはほぼ同じ構造
	
	タンパクファミリー
		例：セリンプロテアーゼのファミリー
			タンパク質分解酵素で、キモトリプシン、トリプシン、エラスターゼ、プロテアーゼなどが該当
			よく似た構造を共有している
	
	タンパク質の高次構造
		相補的結合部位で次々に繋がると、往々にしてらせん構造を作る
		 ... アクチンフィラメントなど
		多くの場合は、材料タンパク質を用意してやると自己組織的に構造を形成する。
		
	球状タンパク質と繊維状タンパク質
		繊維状タンパク質の例 ... αケラチン => 中間径フィラメント
		細胞外マトリックスも繊維状タンパク質で作られる（強固な遷移を形成するコラーゲンなど）
		エラスチンは細胞外マトリックスを構成する不定構造のタンパク質 ... 弾性に寄与
	
	ジスルフィド結合
		システイン同士の結合による構造の安定化をもたらす
		細胞質内では還元的雰囲気なので存在しない
	
	タンパク質の働く仕組み
		特異的な結合（結合する物質をリガンドという）
		例：抗体（antibody） ... 結合部位はペプチドループで形成されているので、配列の微妙な変化で結合性が変わる
		触媒作用（酵素）
		ロドプシンであればレチナール、ヘモグロビンであればヘムのようにタンパク質以外の小分子を持つことも多い
	
	酵素の主な分類
		加水分解酵素
		ヌクレアーゼ : ヌクレオチド間の結合を加水分解
		プロテアーゼ : タンパク質を分解
		合成酵素
		異性化酵素
		ポリメラーゼ
		キナーゼ : リン酸基の付加
		ホスファターゼ : リン酸基の除去
		酸化還元酵素
		ATPアーゼ
	
	例—リゾチーム : 抗生物質（細菌細胞壁の多糖鎖を切断する）
		加水分解を触媒する酵素
		結合している糖のコンフォメーションをゆがめ、不安定化する
		アスパラギン酸の負電荷によって結合を切断
		グルタミン酸の負電荷によってOH基が攻撃して加水分解が完了する
		
	一般に
		1. 構造により安定して結合
		2. 電荷分布の変化
		3. 結合角の変化
		4. 活性化エネルギーの低下
		という過程を考えれば良い。
	
	タンパク質の調整
		1. 発現調節（酵素分子の数の調節）
		2. 物理的隔離
		3. 反応速度の調節
			☆フィードバック阻害（生成物が拮抗剤になる）
	
	アロステリック酵素
		基質とは別の結合部位への第二の物質の結合による調節
		... 酵素のコンフォメーション変化によると思われる
		ATPとタンパクキナーゼによるリン酸化によるコンフォメーション調節はよく使われている
		GTP結合タンパク（GTP-binding protein）
			結合したGTPをタンパク自体が加水分解すると不活性型となる。
			分子スイッチとして細胞内でよく働いている（大きな変化のトリガーとなる）
		ATP加水分解時のエネルギー差によって不可逆反応が生じる
			モータータンパクのように正味の仕事を行うことが出来る
