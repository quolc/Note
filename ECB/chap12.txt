Chap 12. 膜輸送
	
	疎水性の膜を通じて水溶性の栄養分や廃棄物を輸送する必要がある
	膜輸送タンパク質が膜の透過に関与している
	二種類の膜輸送タンパク質
		1. 運搬体タンパク ... 可動部位によって物質を輸送
		2. チャネルタンパク ... 小孔を拡散させることで輸送
	
	膜輸送の原理
		イオン濃度は細胞内と細胞外で大きく異なる
			細胞外：Na+, Cl-が主
			細胞内：K+が主
		脂質二重層は溶質やイオンを通さない
			酸素分子や二酸化炭素など非極性小分子はかなり透過する
			極性分子は水やエタノール程度の小さいものは透過するが、それより大きいとほとんど透過しない
			電荷を持つ分子はほぼ透過しない
		膜輸送タンパク質
			1. 運搬体 : タンパク質の結合部位に特異的に結合する溶質を輸送する
			2. チャネルタンパク質 : 電荷と分子の大きさで透過させる溶質を選別する
		受動輸送と能動輸送
			受動輸送：濃度勾配に従う拡散によるエネルギーを消費しない輸送
			能動輸送：エネルギーを消費して濃度勾配に逆らって移動する輸送 ... チャネルでは行われない
	
	運搬体タンパクとその機能
		運搬対象に関する特異性が高い
		例：バクテリオロドプシン ... 光で活性化するH+ポンプ
		
		濃度勾配と電気的な力による受動輸送
			例：グルコース運搬体
			 ... 少なくとも２つの状態によってランダムに細胞内外にグルコースを移動させる
				 濃度勾配によって均一になるよう分散させることになる
				 選択性は高く、D-グルコース以外とは結合しない
			電気化学的勾配：濃度勾配＋電位差による力
		
		能動輸送
			(1) 共役輸送体 ... ２つの物質の輸送を組み合わせることで濃度勾配に逆らう輸送を実現する
			(2) ATP駆動ポンプ
			(3) 光駆動ポンプ
			これらは連携して動作する。
			例えばNa+をATP駆動ポンプが外部に運搬したものが細胞内に戻る際、共役輸送体によって他の物質の輸送が行われたり
		
		動物細胞でのATP加水分解
			ATP駆動Na+ポンプ ... ATPアーゼの一種と言える
			Na+-K+ポンプのリン酸化により構造を変化させてポンピングを実現する
			Na+の外部輸送とK+の内部輸送を組み合わせて行う
			全ATPの30%程度を使用している
			
			主に細胞膜における浸透圧の調節が大きな役割
		
		共役輸送体
			二種類の溶質が同じ方向に運ばれる - シンポート
			二種類の溶質が逆の方向に運ばれる - アンチポート
			共役ではない輸送 - ユニポート
			
			動物細胞ではNa+と一緒に必要な物質を運び込むシンポートが非常に重要
			アンチポート系ではNa+-H+交換体がpH調節に重要な役割を果たしている
		
		Ca2+ポンプ
			Na+同様に外液に比べて細胞内では小さく保たれている
		
		植物や菌類、最近ではH+勾配が膜輸送の駆動力として用いられる
			植物細胞などではNa+-K+ポンプが存在しないので、代わりにH+ポンプによるH+の電気化学的勾配が用いられる
			光合成細菌のあるグループでは、バクテリオロドプシンのような光駆動H+ポンプでH+を汲み出している
			H+ポンプはこの他にも、液性の差が機能に関わることが多いので重要な役割を果たす
		
	イオンチャネルと膜電位
		能動輸送は出来ない。電気化学的勾配による受動輸送のみを行う
		大半は小さく、イオン選択性を持つ
		基本的にはランダムに開閉する
		ゲートを備え、刺激によって構造を変えて開閉頻度を切り替える
		チャネルでのイオンの流れが急激な膜電位変化を引き起こす
		 -> 電気シグナルの伝達
	
		イオンチャネルの開閉に影響を与える刺激
		(1) 電位依存 ... 膜電位によって調節
		(2) リガンド依存 ... 結合分子によって開閉を調節
		(3) 機械刺激依存 ... 聴覚有毛細胞など
		
		電位依存イオンチャネル
			シグナル伝達に主に機能
			電荷を持った特別なタンパクドメインが電位センサーとして働く
			イオンチャネル -> 膜電位 -> イオンチャネル　という調節回路
			
		膜電位
			静止状態の細胞 ... Na+-K+ポンプの働きでK+が高濃度, またK+漏洩チャネルが存在
			静止膜電位 ... 濃度勾配による流出と電位勾配による流入が釣り合った状態（負電位）
			ネルンストの式で平衡状態での静止膜電位を計算できる
			
			Na+などの他のイオンチャネルが開いた場合
			濃度勾配も電位勾配も流入側に偏っているので、一気に細胞膜内の電位が上昇する
	
	神経細胞のイオンチャネルとシグナル伝達
		神経細胞 - ニューロン
		細胞体, 軸索, 樹状突起で構成される。軸索の先端は神経末端となっている
		樹状突起は他のニューロンからの入力を受け取る
		
		樹状突起や細胞体で受け取ったシグナルを神経末端へと伝達させる ... 信号減衰が大きな問題となる
		増幅のために神経インパルスが発生する（活動電位）
		
		活動電位は電位依存Na+チャネルによって伝播する
			1. 神経伝達物質による局所的脱分極
			2. 閾値に達すると電位依存Na+チャネルが開いて正電荷が流入する
			3. 電位が逆転するまで一気にNa+イオンが流入したあと、電気化学的駆動力がゼロになる
			4. Na+チャネルの不活性化機構が働いてチャネルが閉じる
			5. 膜電位が元の状態に戻る。しばらくチャネルは不活性化状態が続く
			また、電位依存K+チャネルも脱分極に対応して開放され、静止電位への復帰に関与する
		
		電位依存Ca2+チャネルは神経末端で電気シグナルを化学シグナルに変換する
			神経末端に到達した活動電位は標的細胞にシグナル中継する必要がある
			シナプス前細胞とシナプス後細胞がシナプス間隙によって隔てられている
			
			シナプス前細胞のシナプス小包に神経伝達物質が蓄えられている
			活動電位が来ると、電位依存Ca2+チャネルが開放されてCa2+イオンが一気に流入する
			これがきっかけとなってシナプス小包がシナプス前膜と融合してエキソサイトーシスが起こる
			
		標的細胞の神経伝達物質依存チャネルが化学シグナルを電気シグナルに再編関する
			神経伝達物質は神経伝達物質受容体に結合することで活動電位が発生する
			役目を終えた神経伝達物質は直ちに酵素などによって取り除かれる
			受容体 ... 伝達物質依存イオンチャネルが主
			神経伝達物質としてはアセチルコリンが有名
		
		ニューロンが受け取る入力は興奮性と抑制性がある
			興奮性ニューロンは活動電位を発生させる神経伝達物質を放出
			抑制性ニューロンは活動電位を抑制する神経伝達物質を放出
			
			興奮性神経伝達物質 ... アセチルコリン, グルタミン酸など, Na+チャネルなどが受容体
			抑制性神経伝達物質 ... γ-アミノ酪酸（GABA）, グリシンなど, Cl-チャネルなどが受容体
			
		伝達物質依存チャネルは向精神薬の主な標的
			催眠剤やトランキライザー ... GABA依存Cl-チャネルに結合して開きやすくする -> 抑制性の影響を受けやすくなる
			抗鬱剤 ... 興奮性神経伝達物質（セロトニン）が取り込まれるのを阻害する
		
		シナプス接続により思考や行動や記憶が可能になる
