Chap 13. 細胞が食物からエネルギーを得る仕組み
	
	糖と脂肪の分解
		動物細胞は二通りの方法でATPをつくる
			ADP + Pi -> ATP と共役するもの
			ミトコンドリアで起こる反応
		
		食物分子の分解段階
			1. 消化 ... 大型重合分子から単量体に分解される（タンパク質->アミノ酸, 多糖->単糖, 脂肪->脂肪酸＋グリセロール）
			2. 解糖 ... グルコース1分子からピルビン酸2分子を作る, NADHとATPがこの段階で生産される
				ピルビン酸はミトコンドリアでCO2とアセチル基になり、補酵素AとアセチルCoAをつくる
				脂肪酸の分解でも大量のアセチルCoAが生成される
			3. クエン酸回路 ... アセチルCoAからオキサロ酢酸にアセチル基が転移し、クエン酸回路に入る
				電子運搬体NADHを生成し、このエネルギーによってミトコンドリア内膜の電子伝達系でATP生成とO2消費が行われる
				ミトコンドリア内膜でのATP生成を酸化的リン酸化と呼ぶ
			全体のエネルギー効率はおよそ50%程度
		
		解糖glycolysis
			グルコースの分解過程
			グルコース + 2ATP -> 2ピルビン酸 + 4ATP + 2NADH
			NAD+が電子を受け取る形で酸化反応が起こっている
		
		発酵
			嫌気性生物にとっては分子状酸素を用いるミトコンドリアでの反応は使えない
			->解糖がATPを得る主要な手段となる
			ピルビン酸は細胞から排出され、例えばパン酵母ではエタノールと二酸化炭素を生じる
			嫌気的なエネルギー獲得経路を発酵と呼ぶ。
	
		グリセルアルデヒド3-リン酸の酸化過程
			0. グリセリンの分解で二分子のグリセルアルデヒド3-リン酸が生じる
			1. グリセルアルデヒド3-リン酸と脱水素酵素が結合する
			2. NAD+を還元してグリセルアルデヒド3-リン酸のアルデヒド炭素が酸化される
			3. 酵素が脱離し、代わりにリン酸化を受ける
			4. ADPをリン酸化し、ATPを生じる。同時にアルデヒド基はカルボキシル基になる
		全体としてはグリセルアルデヒド3-リン酸が3-ホスホグリセリン酸に酸化され、ATPとNADHがそれぞれ一分子生じる
	
		アセチルCoAの生成
			第三段階ではO2を用いた反応が主となる ... 酸素濃度が十分上昇してから獲得された機構か
			ピルビン酸 -> CO2, NADH, アセチルCoA
			脂肪由来の脂肪酸を分解する酵素もミトコンドリアでアセチルCoAをつくる。
		
		クエン酸回路
			真核生物のミトコンドリアで起こる、アセチルCoAのエネルギーを取り出す回路
			最終生成物はCO2とNADH, NADH->電子伝達系
			アセチル基＋オキサロ酢酸 -> クエン酸 -> ... -> オキサロ酢酸
			
			三種類の運搬体分子
			1. NADH x3
			2. FADH2 x1 ... NADH同様に高エネルギー電子と水素を運搬する
			3. GTP x1 ... ATP生産に用いられる
			
			アミノ酸も一部が細胞質からミトコンドリアに入って反応中間体に変化する
			（すべてのエネルギー獲得がミトコンドリアに集まってくる）
			
			また、炭素中間体はオキサロ酢酸やα-ケトグルタル酸などが生合成の出発点として用いられる
		
		電子伝達系
			ほとんどのATPは電子伝達系における酸化的リン酸化で合成される
			電子受容体と電子供与体が連なっており、電子が順にたどりながらエネルギーを落としていく
			この過程でH+イオンの膜間濃度勾配が生じ、これによってATPポンプが駆動することでATPを生成する
			電子は最後に分子状酸素に移り、H+と結合して水分子になって最低エネルギー状態に落ちつく
			
			1分子のグルコースの酸化によって30ATPが得られるが、このうち解糖系で得られるのはたったの2ATPにすぎない
			
	食物の備蓄と利用
		食物分子をいつでも酸化分解ATP生産できる形で備蓄する必要
		脂肪酸 -> 脂肪滴＠脂肪細胞
		糖 -> グリコゲン（多糖）
		脂肪は分子および重量あたりのエネルギー密度が非常に高く、効率的な備蓄物質である
		
		植物は光合成で生産した糖の一部をデンプンや脂肪に変える
		植物脂肪は動物脂肪と同様にトリアシルグリセロールであるが、脂肪酸の種類が異なる
		どちらも葉緑体に蓄えられる
	
		葉緑体とミトコンドリア
			葉緑体は日光エネルギーを利用してATPとNADPHを作り出し、副産物としてO2を生じる。
			ATPやNADPHは葉緑体内膜内にとどまるので、このエネルギーを用いて糖を作って細胞質に届ける
			糖はミトコンドリアで酸化を受けてATPを生産する
		
	多くの生合成経路は解糖系かクエン酸回路を出発点とする
		アミノ酸、ヌクレオチド、脂質、その他の必須有機小分子の生成が代謝過程で行われる
		細胞内では極めて複雑な代謝反応経路が運用されているが、大きな狂いは生じない
		調節機構が巧妙に酵素生成を調節しているからである
