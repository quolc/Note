Chap 10. 遺伝子とゲノムの進化
	遺伝子の進化 ... 安定性と柔軟性のバランスが重要
	
	遺伝的変動の生成
		生殖系列（germ line） ... 生殖細胞（germ cell）を生み出す細胞系譜 <-> 体細胞（somatic cell）
		体細胞の変異は子孫には伝わらない
	
	遺伝的変化のパターン
		全く新しい遺伝子が作られるということはない。
		既存の塩基配列のコピー＆編集によって新しい遺伝子が構成されていく
		
		1. 遺伝子内変異 ... ヌクレオチドの変化・欠失・重複によるへんか, 複製エラーなど
		2. 遺伝子重複 ... 既存の遺伝子, DNA領域, ゲノム単位で重複が起こる
		3. 遺伝子欠失 ... 染色体の切断が修復されないと起こる
		4. エキソンの混ぜ合わせ ... 別の遺伝子のエキソンどうしが結合される
		5. 遺伝子水平伝播 ... DNA断片が細胞から別の細胞へと移動する
	
	ゲノム変化
		点変異
		修復機構がわずかに見逃す複製のエラー, 細胞分裂のたびに10^-10オーダーの割合で起こる
		細胞世代あたり10^9塩基対に1塩基程度の違い
		選択的中立変異 ... 表現型に影響しない変異, ゲノムに蓄積され続ける
	
	遺伝子ファミリー
		新しい遺伝子を生み出す機構 = 重複と分岐
		遺伝子重複の仕組み
			相同染色体どうしがずれて並ぶ
			相同な反復配列を交点として組み換えが起こる（不等交差）
			長い染色体と短い染色体が出来る
		グロビン遺伝子ファミリーは顕著な重複の痕跡を残している。
			原核生物や原始的な動物では一種類しか存在しないが、魚類、哺乳類と進化を重ねるに連れて種類や振る舞いが高度化した
			重複によって冗長性が確保され、変異へのバッファが効く
		偽遺伝子 ... 遺伝子の重複配列だが、変異によって発現が抑制されているもの
			機能しない遺伝子はすぐに取り除かれるわけではない
	
	エキソン単位の遺伝的変化
		タンパクドメインとエキソンの対応 ... エキソンの繰り返し重複により繰り返しタンパクドメインが作られる
		様々なエキソンの混ぜあわせ ... タンパクドメイン＝部品の組み合わせを行った
	
	転移因子（トランスポゾン）
		転移因子の挿入による直接の遺伝子変化
		転移因子が相同組換えの対象となることにより、ゲノム編成の変化を加速する
		近接領域の転移因子間の領域（エキソン）がまるごと他の遺伝子に挿入されることが多い
	
	遺伝子水平伝播
		細菌ではよく見られる
		特に原始的な環境下でより広く行われたのではないか
		真核生物のセントラルドグマに関する遺伝子が古細菌に似て、代謝に関わる遺伝子が真性細菌に似ているのも説明がつく
		
	生命の系統樹の再構築
		相同遺伝子（homologous gene）を見ることで近縁関係が理解できる
		
		変異と自然選択によって進化は行われる
		有利変化・中立変化はその後子孫に受け継がれる可能性がある
		変異が蓄積しやすいDNA：タンパク質合成にかかわらない、調節の役割もない領域
		必須タンパク質やRNA分子（ハウスキーピング遺伝子） : 大抵の変異が致死的なので保存度が高い
		
		二種の生物種の塩基配列の違いは、別々に進化してきた期間の長さに比例する
			人では、生殖系列が胚から生殖細胞までに200回程度分裂する ... 100塩基程度の変異をきたす
			このようなランダムな変異による進化的広がりを遺伝的浮動という。
		
		ヒトとチンパンジーでは99%近くの配列が同じで、また編成も一部をのぞいて共通している
		Aluレトロトランスポゾン配列の大部分はそれ以前の共通祖先の段階で構成された
	
		シンテニーの保存
			分岐前の遺伝子間の順序関係が、分岐後も同じように隣り合っていること
		
		純化選択 ... 重要な機能を妨げる変異の個体が排除される
		
	ジャンクDNA
		ヒトとトラフグ ... 4億年前に分岐
		ほとんど配列は変わってしまっている分、保存領域は明らか
		トラフグのゲノムは非常に小さいが、遺伝子数は近縁種と殆ど変わらない
		ジャンクDNAを取り除いたと考えられる
	
	生命の系統樹
		rRNA配列を用いるのが一般的
		真性細菌・古細菌・真核生物の３ドメイン
	
	ヒトゲノム
		22本の常染色体と2本の性染色体
		アイデンティティは99.9%程度（1/1000程度の多型）
		タンパク質や構造RNA, 触媒RNAは数％に過ぎない
		平均的な遺伝子サイズが27000塩基対ととても大きい（巨大なイントロンがある）
		3万個程度の遺伝子を持つ
		
		SNP（single-nucleotide polymorphism / すにっぷ）
			一塩基多型, 300万以上存在
			遺伝疾患のマーカーとして機能する
	
		CA反復配列
			複製エラーが起きやすく、反復回数がかなり異なる場合がある
	
		関連生物のDNAとの類似性やシンテニーによって配列の解析（機能予測）ができる
		
		生物間の遺伝子保存度と性質の違いの説明
			材料となるタンパク質はほとんど一緒
			発現調整に用いられる調節DNAにより大きな差がある
			反復配列が特に重要な調節配列として働く, 反復配列は変異を受けやすい
		
		ヒトゲノム解析の難しさ
			選択的スプライシングが多い ... 配列だけを見ても分からない
