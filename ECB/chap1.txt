Chap.1 細胞とは

	生物の３ドメイン
		原核生物
			真性細菌
			古細菌
		真核生物
	
	真核生物のオルガネラ
		核 - 染色体 - 遺伝情報
		ミトコンドリア - 細胞呼吸 - ATP合成
		葉緑体 - クロロフィル - 光合成
		小胞体 ... 細胞外に送り出す物質や細胞膜の成分を合成
		ゴルジ体 ... 小胞体から分子を受け取って修飾
		リソソーム ... 細胞内消化
		ペルオキシソーム ... 過酸化水素の処理
			エンドサイトーシス（飲食作用） : 細胞外から来た物質を小包として取り込んでリソソームで消化
			エキソサイトーシス（開口分泌） : 細胞内小包が細胞膜と融合して内容物を分泌
		細胞質ゾル - リボソーム - タンパク質合成
		
	細胞骨格
		アクチンフィラメント : 細い ... 筋肉
		中間径フィラメント : 中くらい ... 機械的強度
		微小管 : 太い ... 細胞分裂
	
	真核生物の起源
		謎が多い
		恐らく、積極的に運動する捕食者由来ではないか
		運動が激しいので、遺伝子を保護するために核を発展させた？
	
	モデル生物
		真性細菌 ... 大腸菌（E.coli, 4.6Mbp）
		真核生物 ... 酵母（S.cerevisiae）
		植物 ... シロイヌナズナ
		動物 ... ショウジョウバエ（Drosophila melanogaster）、線虫（C.elegans）、マウス、ヒト
	
	遺伝子の共通性
		相同（homologous） ... 別々の生物間で塩基配列が共有されている遺伝子の関係
		
		細菌、古細菌、真核生物で共通の遺伝子は200〜300くらい : 生存するために最低限必要な機能
		最小の遺伝子数の生物 : Mycoplasma genitalium 468個
		
		cdc2を欠損させた酵母に、他の酵母やヒトの等価タンパク質を投入すると分裂機能が回復した
		さらに、ヒトCDC2を導入しても分裂機能が回復した
	
	まとめ
		細胞は生命の基本単位
		細胞・生物は、成長し、エネルギーを変換し、環境に応答し、増殖する
		細胞は細胞膜に包まれている
		細胞はDNAを有し、タンパク質を合成する
		多細胞生物では細胞分化が見られる
		原核生物はオルガネラを持たない。
		原核生物は様々な場所で生息している
		真核生物はオルガネラを持ち、ミトコンドリアや葉緑体など他の生物を取り込んだ痕跡も持つ
